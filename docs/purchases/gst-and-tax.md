# Purchase GST and Tax

The purchase invoice stores input CGST, SGST, IGST, cess, place of supply, reverse-charge state, HSN/SAC, rate, and taxable-value snapshots per line. The Input GST Register reads approved invoice snapshots only.

The register is operational reporting. It is not GST portal reconciliation, e-invoicing, or automated return filing. Exports and filing decisions must be reviewed with an accountant.
