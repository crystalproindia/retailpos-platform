# GST Treatment

SaaS billing delegates CGST, SGST, IGST, cess, and tax-treatment calculation to `GstTaxCalculator` and takes supplier snapshots from `GstSetting`. `gst_document_series` owns financial-year numbering. GST reports must be reviewed with an accountant before filing. This release does not file returns, submit IRNs, or claim e-invoice compliance.
