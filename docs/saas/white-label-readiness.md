# White-label Readiness

Tenant administrators with the `white_label` entitlement can manage display name, existing CMS-media logo/favicon references, primary and secondary colours, support email, sender display name, powered-by visibility, and custom-domain status.

Allowed domain statuses are `not_configured`, `pending`, `verified`, `active`, and `failed`. This foundation does not automate DNS, SSL, custom-domain routing, or public asset rewrites. RetailPOS is the default brand when no tenant settings are present.
